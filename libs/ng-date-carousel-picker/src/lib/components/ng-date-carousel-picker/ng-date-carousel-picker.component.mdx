Styles for story;

```css
ng-date-carousel-picker {
  --dcp-control-button-svg-color: purple;
  --dcp-weekday-color: blue;
  --dcp-month-name-border-color: orange;
  --dcp-month-name-active-background-color: orange;
}
```

You can also rewrite certain variables in whole application, using `:root` selector in your
global `style.(s)css` file. Don't forget to add more importance to value with `!important` rule. For example,

{/* eslint-disable prettier/prettier */}
```css
:root {
  --dcp-control-button-svg-color: purple!important;
  --dcp-weekday-color: blue!important;
  --dcp-month-name-border-color: orange!important;
  --dcp-month-name-active-background-color: orange!important;
}
```

### Localization

You can localize the month names and weekday names. To do that, use the token `DCP_DATE_LOCALE`.
You should use "Unicode locale identifier" as a value ([what Angular uses](https://angular.io/guide/i18n-common-locale-id)).
Default value is `en-US`.

<Canvas of={PickerStories.Localization} />

Provides in story:

```ts
providers: [{ provide: DCP_DATE_LOCALE, useValue: 'es-ES' }],
```
